The main three languages!

HTML5

HTML stands for Hyper Text Markup Language. It was originally designed by a man called Tim Berners-Lee to send research documents from one university to another using what we know now as, The World Wide web.

CSS3

Css3 is mainly used to style the HTML5 documents. Adding colour texture and style to a webpage.

Javascript

Javascript is what gets everthing moving. This could be called the brain of a website as it is used for interaction and behaviour on a website.

The courses and resources that ive used......

Sololearn- I found this a really useful source to learn to code. Sololearn is available for most smartphones which i found really useful as an On The Go Learning Tool. They also have a desktop based site which makes the whole experience customisable to your needs.


Udemy- This is a browser based website ive used mostly on my home pc. I am currently doing a Fullstack Bootcamp Course run and designed by Colt Steele. This is a very instructive and interactive course. Up to now it's the best course i've found so far for learning these three basic languages.


Pluralsight- This is a browser based website ive used mostly on my home pc. I am currently doing a HTML From Scratch course. I've found that Pluralsight is one of the more advanced sites to use and would recommend looking eleswhere if you have no coding knowledge what so ever.

Codeacadamy- This is another browser based learning course which is open to everyone. Even the complete novice like me. This is definately a good place to start if your a complete noob!
